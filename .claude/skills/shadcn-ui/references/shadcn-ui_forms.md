# Forms

Build forms with React and shadcn/ui.

## Pick Your Framework

Start by selecting your framework. Then follow the instructions to learn how to build forms with shadcn/ui and the form library of your choice.

### Available Options

- **React Hook Form** - Build forms in React using React Hook Form and Zod.
- **TanStack Form** - Build forms in React using TanStack Form and Zod.
- **React useActionState** - (Coming Soon)

## Getting Started

The Forms section provides guidance on integrating form management libraries with shadcn/ui components. Choose the framework that best fits your project needs and follow the corresponding documentation to implement robust, type-safe forms in your application.

For detailed implementation examples and best practices, navigate to the specific framework documentation:

- [React Hook Form](/docs/forms/react-hook-form)
- [TanStack Form](/docs/forms/tanstack-form)
