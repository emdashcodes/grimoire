# Skeleton

Use to show a placeholder while content is loading.

## Installation

```bash
pnpm dlx shadcn@latest add skeleton
```

## Usage

```javascript
import { Skeleton } from "@/components/ui/skeleton"
```

```jsx
<Skeleton className="h-[20px] w-[100px] rounded-full" />
```

## Examples

### Avatar

Display a skeleton placeholder shaped like an avatar while loading user profile images or icons.

### Card

Show skeleton loading states for card components with header and content areas.

### Text

Skeleton placeholders for text content during the loading phase.

### Form

Skeleton elements for form fields and input areas while form data loads.

### Table

Skeleton rows and cells for table components during data fetching.
