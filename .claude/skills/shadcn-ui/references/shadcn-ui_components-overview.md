## Alternative Implementations

The same components are also available through Radix UI implementations, providing different underlying libraries while maintaining consistent APIs.

## Additional Resources

The registry directory contains community-maintained components for additional needs beyond the core library. The documentation provides comprehensive guides for installation, theming, and integration with various frameworks including Next.js, Vite, Laravel, React Router, Remix, Astro, and TanStack Start.
