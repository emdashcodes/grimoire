# Toast

A succinct message that is displayed temporarily.

## Deprecation Notice

The Radix UI toast component has been deprecated. Use the [Sonner](/docs/components/sonner) component instead.

## Variants

- **Radix UI**: Deprecated. Use Sonner instead.
- **Base UI**: Base UI also provides a toast component option.
