# Distributing Apps With Electron Forge

Electron Forge is an all-in-one tool for packaging and distributing Electron applications. It unifies Electron's build tooling ecosystem into a single extensible interface so that anyone can jump right into making Electron apps.

## Getting started

The Electron Forge docs contain detailed guides on taking your application from source code to your end users' machines. This includes:

- Packaging your application
- Generating executables and installers for each OS platform
- Publishing to online platforms such as GitHub Releases

For beginners, we recommend following Electron's tutorial to develop, build, package, and publish your first Electron application. If you already have an existing app and want to get started with Forge, try Forge's importing guide.

## Alternative tools

If Electron Forge does not fit your needs, there are also other community-maintained packaging and distribution tools.

### electron-builder

Electron Builder is a complete solution to package and build a ready-for-distribution Electron app that is focused on an integrated experience. It replaces features and modules used by the Electron maintainers (such as the auto-updater) with custom ones.

### Hydraulic Conveyor

Hydraulic Conveyor is a desktop app deployment tool made by a commercial company. It can build apps for every desktop platform from any platform (e.g. you can make Windows and Mac apps from Linux), provides synchronous web-style update system, and handles code signing and uploading for you. It's free for open source apps.
