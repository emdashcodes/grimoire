# Adding Features

## Adding Application Complexity

If you have been following along with the tutorial, your application should have a functional Electron application with a static user interface. From this starting point, you can generally progress your app development in two broad directions:

1. **Adding complexity to your renderer process web code** -- Building sophisticated UI using standard web technologies (HTML, CSS, JavaScript). Electron's resources are similar to those for normal web development. Much of the tooling and paradigms that you use for web development can be used in Electron directly.

2. **Deepening your integration with the operating system and Node.js** -- Leveraging Electron's desktop capabilities beyond a regular web page, including tray icons, global shortcuts, native menus, and more.

Ultimately, you are building your renderer's UI using the same tools (HTML, CSS, JavaScript) that you would on the web. Therefore, Electron's documentation does not cover standard web development topics in depth and instead focuses on desktop-specific integrations.
